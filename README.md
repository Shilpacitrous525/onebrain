## 🚀 What onebrain does

onebrain is a local app that helps you keep AI context in one place. It stores notes, prompts, and memory so you can use the same context across ChatGPT, Claude, Gemini, and other tools. It also offers a REST API, MCP support, and SDKs for users who want to connect other apps later.

It is built for people who want private, self-hosted memory that stays under their control. You can run it on your own Windows PC and keep your data local.
